# Sujal Nayak — Portfolio

Personal portfolio website showcasing projects, skills, experience, and certificates.

**Live site:** [sujalnayakdotcom.github.io](https://sujalnayakdotcom.github.io)

## Features

- Animated page loader and custom cursor
- Responsive navigation with mobile hamburger menu
- Hero section with profile image and scrolling marquee
- Projects showcase with hover effects
- About section with animated stat counters
- Experience timeline
- Skills displayed in a bento grid layout
- Certificate slider with touch swipe support and dot indicators
- Contact form with mailto/Gmail integration and WhatsApp link
- Scroll-reveal animations via Intersection Observer

## Tech Stack

- **HTML5** / **CSS3** / **JavaScript** (vanilla, no frameworks)
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) & [Space Mono](https://fonts.google.com/specimen/Space+Mono) fonts via Google Fonts
- Hosted on **GitHub Pages**

## Project Structure

```
├── index.html          # Main HTML page
├── style.css           # All styles (responsive, animations, layout)
├── script.js           # Interactions (cursor, slider, scroll reveals, form)
└── assets/             # Images, certificates (PDF + JPG thumbnails)
```

## Running Locally

Open `index.html` in a browser, or serve with any static file server:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .
```
